# Ezyride   Electric Scooty Sales and Services App

## Overview
A web application for electric scooter sales and services under the brand "Ezyride" with tagline "Bhavishya ki Sawari" and virtual showroom "MBEvHub". The application allows customers to browse models, make purchases, book test rides, and schedule service appointments. Includes an admin panel for managing all operations.

## Branding
- **Brand Name:** Ezyride
- **Tagline:** Bhavishya ki Sawari
- **Showroom Name:** MBEvHub
- **Language:** English

## Core Features

### Homepage & Product Catalog
- Homepage hero section prominently displays "Ezyride" brand name and "Bhavishya ki Sawari" tagline
- Display available electric scooter models with images, specifications, and pricing
- Each model shows key details like range, top speed, charging time, and price
- Product filtering and search functionality
- Footer displays brand name and tagline prominently

### Online Purchasing
- Shopping cart functionality to add selected scooter models
- Integrated checkout flow with customer information collection
- Order confirmation and tracking
- MBEvHub showroom branding displayed throughout purchase flow

### Test Ride Booking
- Calendar-based booking system for test rides at MBEvHub virtual showroom
- Users can select preferred scooter model, date, and time slot
- Booking confirmation with details
- MBEvHub showroom name displayed on test ride pages

### Service Appointment Scheduler
- Appointment booking for maintenance and battery replacement services at MBEvHub
- Service type selection (maintenance, battery replacement, repairs)
- Date and time slot selection
- Customer contact information collection
- MBEvHub showroom name displayed on service booking pages

### Contact Us Page

- Map displays precise location marker for the showroom
- Contact form below the map for customer inquiries
- Page maintains Ezyride branding with logo, colors, and tagline alignment

### Admin Panel
- Dashboard displays "MBEvHub" as the showroom name
- Manage scooter model listings (add, edit, delete models with specifications and pricing)
- View and manage purchase orders
- View and manage test ride bookings
- View and manage service appointments
- Dashboard with overview of recent activities

- Scooter models with specifications, pricing, and images

- Admin user authentication data
